# 音乐资产

游戏使用两首全局背景音乐，并按以下顺序循环播放：

1. `shan-he-wei-ding-v1.mp3`：《山河未定》，古琴、洞箫、琵琶、低鼓、编钟和少量二胡构成的克制中国风氛围。
2. `feng-huo-chang-ge-v3.mp3`：《烽火长歌》v3，以 v2 为参考音频进行改编，仅用琵琶和古筝表现苍茫的历史回望与战争史诗感，并明确排除任何人声。

当存在待布阵、交战中或待结算的实时战斗时，全局歌单会临时切换为 `jin-ge-po-zhen-v1.mp3`《金戈破阵》。该曲以琵琶和中国大鼓表现军阵推进、冲锋与反击，战斗期间单曲循环；战斗完成后恢复全局双曲目歌单。

三首运行时曲目均使用 MiniMax 生成。《山河未定》与《金戈破阵》使用 Music 2.6 纯器乐模式；《烽火长歌》v3 使用 `music-cover` 参考 v2 改编。`feng-huo-chang-ge-v1.mp3` 与 `feng-huo-chang-ge-v2.mp3` 均已停用，仅保留作版本追溯。

运行时播放与战斗切换规则由 `godot/scripts/Main.cs` 定义。文件不得被静默覆盖；后续修改使用 `-v2`、`-v3` 等版本后缀，并同步更新资产清单与 `docs/05-generated-assets.md`。
